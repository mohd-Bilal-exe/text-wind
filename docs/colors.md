# Colors

Use the `text-*-color` utilities for typography color tokens:

- `text-heading-color`
- `text-subheading-color`
- `text-body-color`
- `text-muted-color`
- `text-accent-color`

Opacity modifiers work the same way as Tailwind color utilities:

```tsx
<p className="text-body-color/80">Muted body text</p>
```
