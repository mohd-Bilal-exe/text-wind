# Font Families

Use the `font-*` utilities to keep typography tokens consistent:

- `font-display`
- `font-heading`
- `font-subheading`
- `font-title`
- `font-body`
- `font-label`
- `font-callout`
- `font-caption`
- `font-overline`
- `font-footnote`
- `font-mono`
- `font-code`

Each utility maps to a CSS variable by default, so you can re-theme the system from `@layer theme`.
