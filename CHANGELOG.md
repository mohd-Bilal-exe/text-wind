# Changelog

## 1.6.0

System-wide typography expansion with new architectural categories.

- **New Typography Types**: Added `display`, `title`, `label`, `overline`, `callout`, `footnote`, and `code` as standard fluid types.
- **Enhanced Font Tokenization**: Expanded default font family mappings for more granular control.
- **New Color Utilities**: Added `text-callout-color` and `text-muted-color` for specialized text states.
- **Documentation Overhaul**: Updated the reference section and markdown docs to reflect the new system registry.

## 0.1.0

Initial release of `fluidwind`.

- Fluid typography utilities with inline scale modifiers
- Font family utilities
- Color utilities with opacity modifiers
- Optical letter-spacing scaling
- Tailwind v3.4+ and v4.x compatible package scaffold
